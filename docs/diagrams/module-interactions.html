ЛИСТ 02

Внутреннее устройство gui/

View рисует и передаёт команды. Состояние решает, что изменилось. Services знают формат ядра, но не знают DearPyGui.

composition root view/app.py · PVTcalcApp ViewContext(state, session, jobs)

Создаёт DPG context/layout, связывает feature-view, подписывается на StateChange, сохраняет сессию и опрашивает job.

feature views projects / composition / flash / experiment / envelope + workspace / dialogs / new_fluid_form

Каждый экран импортирует DPG, свой GUI-сервис и контракт ViewHost; обратного импорта из feature-view в app.py нет.

view model app_state.py · AppState Project / Model / Variant / GraphNode / NodeRef

Команды редактирования, навигация, dirty/save, undo/redo, история узлов и ленивая инвалидация; Compare хранит полные межмодельные NodeRef.

events StateChangeKind FULL · NAVIGATION · MODEL_LIST · TREE · WORKSPACE · NODE

Определяет минимальную область перерисовки. NODE-событие несёт NodeRef, поэтому соседние вкладки не пересоздаются.

calculations calculation_coordinator.py CalculationJob(node_ref, result, error, done)

Не более одной daemon worker-задачи. View применяет результат в главном DPG-потоке к адресу model/variant/node.

persistence session.py + workspace_codec.py SessionState ↔ gui_session.json v3

Сохраняет окно, workspaces, параметры и снимки результатов; санирует повреждения, RUNNING после рестарта становится STALE.

comparison R3.2 services/comparison_service.py DLE / CCE / Separator → series + deviations

Без DearPyGui проверяет одинаковый тип и условия, сопоставляет сетки, готовит межмодельные расчётные/Lab Data кривые и абсолютные/относительные отклонения.

Текущий feature-view контракт реализован mixin-композицией через ContextBoundView. Переход к объектной композиции возможен по одному экрану и не требует менять services/state.

канонический основной PhaseEnvelopeSuccessiveSubstitution.py PhaseEnvelopeSSM

Основной GUI-путь кривой огибающей.

канонический точечный BubblePointPressure.py / DewPressure.py BubblePointCalculator / DewPointCalculator

Сырой converged=True недостаточен: фасад подтверждает смену устойчивости и физический тип.

канонический альтернативный PhaseEnvelopeNewton.py / PhaseEnvelopeGrid.py PhaseEnvelopeNewton / PhaseEnvelopeGrid

Newton — уточнение точек; Grid — карта стабильности и затравки.

диагностика R1.1 Diagnostics.py EnvelopeDiagnostic / attach_diagnostics

Различает штатное отсутствие ветви, несходимость, неверифицированный корень и пропуск после остановки; GUI получает тот же JSON-контракт.

не показывается в GUI CriticalPoint.py CriticalPointCalculator

Единственная живая публичная реализация, но пока недостоверна для многокомпонентных составов.

переходный путь new_methodv2.py SaturationPressure

Не используется GUI-огибающей, но пока вызывается DLE/CCE/Separator.

изолировано *_den_v.py + CVD_den_v.py альтернативная критическая точка / незавершённый CVD

Не входят в фасад и GUI; CVD имеет сломанный импорт класса.

R1.1: полная огибающая — SSM; Newton — независимая сверка с fallback на SSM; Grid — карта, а не точная Psat. Причина NaN больше не теряется. Ноутбучные реализации «From VBA» / «Parallel» удалены 2026-07-19.
